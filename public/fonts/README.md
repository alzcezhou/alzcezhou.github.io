# Fonts

1. Unzip `Jacquard_24,VT323,Vollkorn.zip` (from your Desktop).
2. Copy the font files into this folder and name them as below (rename if needed):

   - **Jacquard (display):** `Jacquard24-Regular.ttf` — from the Jacquard_24 folder
   - **VT323 (accent):** `VT323-Regular.ttf`
   - **Vollkorn (body/headings):** `Vollkorn-Regular.ttf` (and optionally `Vollkorn-Italic.ttf`)

If your zip has different filenames (e.g. `Vollkorn-VariableFont_wght.ttf`), either rename them to match the names above or update the paths in `src/styles/global.css` in the `@font-face` rules.
